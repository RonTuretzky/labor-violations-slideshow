# Labor Rights Violations — Research Slideshow

A password-protected static slideshow on the most common and highest-impact labor
violations worldwide (wage theft, forced labor, child labor, safety failures, and more),
with per-slide note boxes and a copy-all-notes export.

All 19 underlying source claims passed a 3-vote adversarial fact-check (19/19 confirmed).
Sources: ILO, EPI, Human Rights Watch, WHO/ILO, EU Forced Labour Regulation analysis.

The page is encrypted with [StatiCrypt](https://github.com/robinmoisson/staticrypt).
